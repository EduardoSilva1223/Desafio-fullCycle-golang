# Desafio-fullCycle-golang


## criar o build da imagem 


```
docker build -t yaboku/golang-cycle:latest .  
```

## Realizar o run da imagem 


```
docker run --rm yaboku/golang-cycle:latest
```

docker push yaboku/golang-cycle:latest
